## –®–∞–≥ 1: –ó–∞–≥—Ä—É–∂–∞–µ–º HTML-—Å—Ç—Ä–∞–Ω–∏—Ü—É —Å —Å–µ—Ä–≤–µ—Ä–∞ –∏ –∏–∑–≤–ª–µ–∫–∞–µ–º —É—Ä–∞–≤–Ω–µ–Ω–∏–µ

–î–ª—è –Ω–∞—á–∞–ª–∞ –Ω—É–∂–Ω–æ –∑–∞–≥—Ä—É–∑–∏—Ç—å HTML-—Å—Ç—Ä–∞–Ω–∏—Ü—É —Å –≤–µ–±-—Å–µ—Ä–≤–µ—Ä–∞, –Ω–∞ –∫–æ—Ç–æ—Ä–æ–π –Ω–∞—Ö–æ–¥—è—Ç—Å—è —É—Ä–∞–≤–Ω–µ–Ω–∏—è, –∏ –∏–∑–≤–ª–µ—á—å –Ω—É–∂–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ (—É—Ä–∞–≤–Ω–µ–Ω–∏–µ –∏ –∏–Ω–¥–µ–∫—Å —É—Ä–∞–≤–Ω–µ–Ω–∏—è).

    ```python
    z = requests.get('http://127.0.0.1:6969').text
    ```

2. **–ü–∞—Ä—Å–∏–Ω–≥ HTML**: –ò—Å–ø–æ–ª—å–∑—É–µ–º –±–∏–±–ª–∏–æ—Ç–µ–∫—É `BeautifulSoup` –¥–ª—è –ø–∞—Ä—Å–∏–Ω–≥–∞ HTML-–∫–æ–¥–∞ —Å—Ç—Ä–∞–Ω–∏—Ü—ã.


    ```python

    ```

4. **–ò–∑–≤–ª–µ—á–µ–Ω–∏–µ —É—Ä–∞–≤–Ω–µ–Ω–∏—è**: –£—Ä–∞–≤–Ω–µ–Ω–∏–µ —Ö—Ä–∞–Ω–∏—Ç—Å—è –≤ —ç–ª–µ–º–µ–Ω—Ç–µ `<p>`. –ú—ã –Ω–∞—Ö–æ–¥–∏–º —ç—Ç–æ—Ç —ç–ª–µ–º–µ–Ω—Ç –∏ –∏–∑–≤–ª–µ–∫–∞–µ–º —Ç–µ–∫—Å—Ç —É—Ä–∞–≤–Ω–µ–Ω–∏—è.

## –®–∞–≥ 2: –†–µ—à–∞–µ–º —É—Ä–∞–≤–Ω–µ–Ω–∏–µ —Å –ø–æ–º–æ—â—å—é SymPy

–¢–µ–ø–µ—Ä—å, –∫–æ–≥–¥–∞ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –∏–∑–≤–ª–µ—á–µ–Ω–æ, –µ–≥–æ –Ω—É–∂–Ω–æ —Ä–µ—à–∏—Ç—å. –î–ª—è —ç—Ç–æ–≥–æ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –±–∏–±–ª–∏–æ—Ç–µ–∫–∞ `SymPy`.

1. **–ü—Ä–µ–æ–±—Ä–∞–∑—É–µ–º —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –≤ —Å–∏–º–≤–æ–ª—å–Ω–æ–µ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ**:
   - –°–Ω–∞—á–∞–ª–∞ –¥–µ–ª–∏–º —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –Ω–∞ –ª–µ–≤—É—é –∏ –ø—Ä–∞–≤—É—é —á–∞—Å—Ç–∏ –ø–æ –∑–Ω–∞–∫—É —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ `"="`.
   
    ```python
    lhs, rhs = equation_str.split('=')
    ```

2. **–ö–æ–Ω–≤–µ—Ä—Ç–∏—Ä—É–µ–º —Å—Ç—Ä–æ–∫–∏ –≤ —Å–∏–º–≤–æ–ª—å–Ω—ã–µ –≤—ã—Ä–∞–∂–µ–Ω–∏—è**:
   - –ò—Å–ø–æ–ª—å–∑—É–µ–º `sp.sympify()` –¥–ª—è –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è –ª–µ–≤–æ–π –∏ –ø—Ä–∞–≤–æ–π —á–∞—Å—Ç–∏ —É—Ä–∞–≤–Ω–µ–Ω–∏—è –≤ —Å–∏–º–≤–æ–ª—å–Ω—ã–µ –≤—ã—Ä–∞–∂–µ–Ω–∏—è.

    ```python

    rhs = sp.sympify(rhs.strip())
    ```

3. **–°–æ–∑–¥–∞–µ–º —Å–∏–º–≤–æ–ª—å–Ω–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ**:
   - –°–æ–∑–¥–∞–µ–º —É—Ä–∞–≤–Ω–µ–Ω–∏–µ —Å –ø–æ–º–æ—â—å—é `sp.Eq()`.

4. **–†–µ—à–∞–µ–º —É—Ä–∞–≤–Ω–µ–Ω–∏–µ**:


5. **–ß–∏—Å–ª–µ–Ω–Ω–æ–µ —Ä–µ—à–µ–Ω–∏–µ**:
   - –ï—Å–ª–∏ –Ω–µ —É–¥–∞–µ—Ç—Å—è –Ω–∞–π—Ç–∏ –∞–Ω–∞–ª–∏—Ç–∏—á–µ—Å–∫–æ–µ —Ä–µ—à–µ–Ω–∏–µ, –∏—Å–ø–æ–ª—å–∑—É–µ–º —á–∏—Å–ª–µ–Ω–Ω—ã–π –º–µ—Ç–æ–¥ `sp.nsolve()`.

    ```python
    if not solutions:
        solution_numerical = sp.nsolve(equation, x)
        return [solution_numerical]
    ```

–ü–æ—Å–ª–µ —Ç–æ–≥–æ –∫–∞–∫ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ —Ä–µ—à–µ–Ω–æ, –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –æ—Ç–ø—Ä–∞–≤–∏—Ç—å —Ä–µ–∑—É–ª—å—Ç–∞—Ç –æ–±—Ä–∞—Ç–Ω–æ –Ω–∞ —Å–µ—Ä–≤–µ—Ä.
